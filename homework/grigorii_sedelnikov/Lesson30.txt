<  ESC-<   
	Go to first line in file
>  ESC->
	Go to last line in file
^F  ^V  SPACE
	Forward one window
^E  j  ^N  CR
	Forward one line
^Y  k  ^K  ^P
	Backward one line
/pattern     
	Search forward for matching line
?pattern  
	Search backward for matching line
 n
	Repeat previous search
N
	Repeat previous search in reverse direction
